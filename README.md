# 콘솔 기반 실시간 커머스 플랫폼 (Java)

Java 기본 문법과 객체지향 설계를 바탕으로  
콘솔 환경에서 동작하는 커머스 플랫폼을 구현.

---

## 주요 기능

### 1) 카테고리 & 상품 조회
- 메인 메뉴에서 카테고리(전자제품/의류/식품) 선택
- 카테고리 내 상품 목록 조회
- 가격 필터링 기능
  - 전체 상품 보기
  - 100만원 이하 상품 보기
  - 100만원 초과 상품 보기
- 상품 선택 후 상세 정보 확인

### 2) 장바구니
- 상품을 선택하고 수량 입력 후 장바구니에 추가
- 장바구니 확인(내역/총액 출력)
- 장바구니 주문 취소(비우기)
- 장바구니에서 특정 상품 제거 (stream.filter 활용)

### 3) 주문 & 재고 관리
- 주문 확정 시 장바구니 항목을 기준으로 총액 계산
- 주문 완료 시 상품 재고 차감
- 재고 부족 시 장바구니 추가 불가 처리

### 4) 고객 등급 & 할인 (Enum)
- 고객 누적 주문금액에 따라 등급 산정
- Enum 기반 할인율 적용
  - BRONZE / SILVER / GOLD / PLATINUM 등급별 할인

### 5) 관리자 모드
- 비밀번호 인증 (3회 실패 시 메인 복귀)
- 상품 추가 / 수정 / 삭제
- 전체 상품 현황 조회

---

## 설계 구조(클래스)
- `Main` : 프로그램 시작점, 초기 데이터 생성
- `CommerceSystem` : 메인 흐름/입력 처리/메뉴 제어
- `Category` : 카테고리 이름 + 상품 리스트 관리
- `Product` : 상품 정보(이름, 가격, 설명, 재고) + 재고 차감 기능
- `Customer` : 고객 정보 + 누적 주문금액/등급 관리
- `Cart`, `CartItem` : 장바구니 관리(추가/조회/비우기/삭제)
- `AdminSystem` : 관리자 인증 및 상품 관리 기능
- `ProductFilter` : 스트림 기반 상품 필터링 및 출력
- `InputUtil` : 숫자 문자 입력

---

## 실행 방법
1. 프로젝트 실행
2. `Main` 클래스 실행 (`public static void main`)
3. 콘솔 메뉴에 따라 숫자를 입력하여 사용
